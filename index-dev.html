Kreamer - Grunt Quick Start

This is a simple HTML document that is based on Twitter Bootstrap. This file loads each of the CSS and JavaScript development files.

The production version of this HTML is located index-prod.html, which loads the minified versions of the CSS and JavaScript files. In a production environment there would be only one HTML file with the application server dynamically switching between the non-minified development files and the minified production files based on the environment.
